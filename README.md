 매일 컴퓨터 사용을 하는 김에 할게 없을까 일기라도 작성하자는 생각에만든
 
 Gemini 1.5 Flash와 Pollinations.ai를 활용한 지능형 감성 다이어리 프로젝트입니다.

---

## ✨ Key Highlights

* **🌈 Multi-Modal AI Analysis**: 일기 텍스트를 기반으로 **감정(Emoji)**, **이미지(Visual)**, **위로(Text)** 세 가지 결과를 동시에 도출합니다.
* **💎 Glassmorphism Design**: 밤하늘 테마의 깊이감 있는 투명 카드 레이아웃을 구현했습니다.

---

## 🛠 Tech Stack

| 분류 | 기술 스택 |
| :--- | :--- |
| **Frontend** | React, Vite, CSS (Glassmorphism), React-Icons |
| **Backend** | Python, Django, Django REST Framework |
| **AI Model** | Google Gemini 1.5 Flash, Pollinations.ai |
| **Database** | SQLite (Development) |
